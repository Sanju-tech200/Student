# Studentcrud

A React + Vite application for managing student records with CRUD operations.

## Tech Stack

- **Frontend Framework**: React
- **Build Tool**: Vite
- **Styling**: CSS
- **Backend**: Firebase
- **Deployment**: Firebase Hosting

## Features

- Create, Read, Update, and Delete student records
- Student form component for data entry
- Student list display with management options
- API integration for backend operations

## Getting Started

### Prerequisites

- Node.js and npm installed
- Firebase project configured

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
